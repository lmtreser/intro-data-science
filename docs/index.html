Anlisis de Datos "Sueldos IT en Argentina"

Lucas Martn Treser

Analisis en base a los resultados de la Encuesta de sueldos 2023.2 por el grupo "SysArmy".

Tema a abordar

Sysarmy es una comunidad de sistemas que nuclea a profesionales del rea para favorecer el contacto y el intercambio de conocimiento de manera informal. Desde el 2014 realizan una encuesta annima para saber cmo se encuentran los sysadmins (y otros profesionales) en su ambiente laboral, a partir de la cual se desprenden distintos analisis de datos para brindar informacin sobre la industria IT en Argentina.

Bsqueda de datos

Los datos que voy a utilizar corresponden a un dataset que se encuentra disponible en el siguiente enlace. El dataset refiere a una encuesta annima donde se relevan datos tanto salariales como de condiciones laborales en el sector IT de nuestro pas.

Carga e informacin general del Dataset

Para trabajar en el analisis voy a utilizar la biblioteca Pandas junto a otras vistas en clase. Pandas es una de las bibliotecas ms utilizadas para analizar y manipular datos. Nos permite trabajar con una nueva estructura de datos, los Data Frame, que permite visualizar los datos en forma de tabla. Adems proporciona una gran cantidad de funciones para manipular y transformar los Data Frame de forma sencilla, rpida y cmoda, proporcionando herramientas para poder leer datos y convertirlos en Data Frames de forma automtica.
